# <a id="config"></a>⚙ » Config

Si vous souhaitez modifier la configuration, ouvrez [config.json](https://discord.gg/akamfr) et localisez-le en haut. Là, vous pouvez configurer les éléments suivants :

```js
  "token": "Put-Token-Here",
  "prefix": "!",
  "color": "#2b2d31",
  "nsfwChannel": true
```
